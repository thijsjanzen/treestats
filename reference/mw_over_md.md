# Maximum width of branch depths divided by the maximum depth

Calculates the maximum width divided by the maximum depth.

## Usage

``` r
mw_over_md(phy)
```

## Arguments

- phy:

  phylogeny or ltable

## References

C. Colijn and J. Gardy. Phylogenetic tree shapes resolve disease
transmission patterns. Evolution, Medicine, and Public Health,
2014(1):96-108, 2014. ISSN 2050-6201. doi: 10.1093/emph/eou018.
