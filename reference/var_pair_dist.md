# Variance of all pairwise distances.

After calculating all pairwise distances between all tips, this function
takes the variance across these values.

## Usage

``` r
var_pair_dist(phy)
```

## Arguments

- phy:

  phylo object or ltable

## Value

Variance in pairwise distance

## References

Webb, C., D. Ackerly, M. McPeek, and M. Donoghue. 2002. Phylogenies and
community ecology. Annual Review of Ecology and Systematics 33:475-505.
