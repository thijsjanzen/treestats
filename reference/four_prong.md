# Four prong index

Calculate the number of 4-tip caterpillars.

## Usage

``` r
four_prong(input_obj)
```

## Arguments

- input_obj:

  phylo object or ltable

## Value

number of 4-tip caterpillars

mean and variance of the numbers of r-pronged nodes and r-caterpillars
