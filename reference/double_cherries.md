# Double Cherry index

Calculate the number of double cherries, where a single cherry is a node
connected to two tips, and a double cherry is a node connected to two
cherry nodes.

## Usage

``` r
double_cherries(input_obj)
```

## Arguments

- input_obj:

  phylo object or ltable

## Value

number of cherries

## References

Chindelevitch, Leonid, et al. "Network science inspires novel tree shape
statistics." PloS one 16.12 (2021): e0259877.
