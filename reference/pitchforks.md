# Number of pitchforks

Pitchforks are a clade with three tips, as introduced in the phyloTop
package.

## Usage

``` r
pitchforks(input_obj, normalization = "none")
```

## Arguments

- input_obj:

  phylo object or ltable

- normalization:

  "none" or "tips", in which case the found number of pitchforks is
  divided by the expected number.

## Value

number of pitchforks
