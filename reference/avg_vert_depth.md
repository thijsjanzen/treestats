# Average vertex depth metric

The average vertex depth metric, measures the average path (in edges),
between the tips and the root.

## Usage

``` r
avg_vert_depth(phy)
```

## Arguments

- phy:

  phylo object or ltable

## Value

Average depth (in number of edges)
