# Mean Nearest Taxon distance

Per tip, evaluates the shortest distance to another tip, then takes the
average across all tips.

## Usage

``` r
mntd(phy)
```

## Arguments

- phy:

  phylo object or ltable

## Value

Mean Nearest Taxon Distance.

## References

Webb, C., D. Ackerly, M. McPeek, and M. Donoghue. 2002. Phylogenies and
community ecology. Annual Review of Ecology and Systematics 33:475-505.
